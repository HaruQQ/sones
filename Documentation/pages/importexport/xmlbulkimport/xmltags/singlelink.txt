====== SingleLink tag ======

===== Description =====
Represents a single-edge. It contains one [[importexport:xmlbulkimport:xmltags:Link|Link]] tags that will be stored under the outgoing edge attribute, that is referenced with [[importexport:xmlbulkimport:xmlattr:Key|AttributeName]].
