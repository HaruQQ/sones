====== Insert tag ======

===== Description =====
Represents one vertex to be inserted. It is allowed to have multiple insert tags in an [[importexport:xmlbulkimport:xmltags:Import|Import]] tag. The attribute [[importexport:xmlbulkimport:xmlattr:VertexType|VertexType]] assigns the vertex type and the attribute [[importexport:xmlbulkimport:xmlattr:VertexID|VertexID]] assigns the vertex id of the new vertex. Both attributes are mandatory and can be used to reference this vertex in a [[importexport:xmlbulkimport:xmltags:Link|Link]] tag.
